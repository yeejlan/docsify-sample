# docsify-sample

### Quick start

npm i docsify-cli -g

docsify serve docs

### Run in docker

docker build -f Dockerfile -t docsify/demo .

docker run -itd -p 3000:3000 -v Path_to_this_repo\docsify-sample\docs:/docs --name demo-docs docsify/demo
